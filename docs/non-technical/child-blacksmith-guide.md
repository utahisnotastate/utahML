# The Magical Code Blacksmith (Kid-Friendly Guide)

Imagine your project is a castle full of robot builders.

- Each file is a room.
- Each function is a machine.
- Sometimes a machine is unfinished.

## The Magic Spell

Inside a function, you can write:

` ; wishes: make this machine count from 1 to 10 `

When you save, the invisible blacksmith:

1. reads your wish,
2. checks other rooms so styles match,
3. builds the machine,
4. leaves a safety stamp:
   - `# [UTAH_HARDCORE_COMPLIANCE_PASS]`

## Tiny Example

```python
def count_to_ten() -> list[int]:
    ; wishes: return numbers from 1 to 10 in order
    pass
```

Then the blacksmith helps finish it.

## Safety Rule

Always ask a grown-up engineer (or your teacher) to review generated code before using it in important projects.
